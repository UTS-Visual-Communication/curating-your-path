For This Task

This brief aims to be a basic introduction to the web, how the web works and how text and image work online. We will also introduce you to your own web space that you will be able to use throughout your Visual Communication degree. Sometimes you might use this for course work. But you are also free to use this as a play space!

Over the course of the rest of this semester, you will be working towards bringing the curated narrative of your year's work together into a website. This website will be designed to have your work as its main focus, with a well thought through grid underlying its design and typographic and visual heirachy that emphasis the most important aspects of your work.

We want to you to consider the typographic detailing (font, colour, size, spacing), the visual hierarchy and the combination of text and image on your web page. As we progress you will also implement a basic grid which will help to define more complex layouts for your pages.

The webpage you are currenltly looking at is made from a single HTML file, with almost no CSS styling, but as you can see its typograpchic heirachy is well defined. As you will see this is because of the inbuilt heirachy of the HTML heading tags "h1, h2, h3, h4, h5, & h6". The code for this page can be "inspected" using your browsers inspector, or "view source".

Session 1—basic HTML, CSS & typographic heirachy

download

Download this weeks code files here.

You will be guided through these files in the studio session.

1.1.1 - 1.1.4 guide you throught the basics of HTML

  • 1.1.1 is a bareboned HTML file, the aboslute minimum you need for a standards compliant web page
  • 1.1.2 guides you through the basic tags you will need to know in order to build a basic web page. These tags form the basis of every site you see on the web.
  • 1.1.3 Is an example of using the 'h' (heading) tags to create heirachy in a basic text.
  • 1.1.4 Is a template of a page with text that has no tags providing typographic heirachy. Your task is to work with file to add a meaningful heirachy to the text.

1.2.1 - 1.2.3 guide you throught the basics of CSS

  • 1.2.1 looks at how to use CSS to style your text. The outcome is not pretty, but it is a starting place for you to play with CSS
  • 1.2.2 Looks at using Google web fonts in your pages. You can use the generic web safe fonts (i.e. ariel, times, verdana). But you should also aim to use web fonts. As they are free and accessible we will be using Google Web Fonts in class and recommend you stick to them for your assignment (for now!)
  • 1.2.3 Use this as a basis for creating a webpage for one of your projects you want to include in your folio - keep it simple.

Working on Your Design

You should use 1.2.3-Basic-Layout as a basis for a page to display one of your projects from this year. You can work directly with the code in order to achieve your design. Sometimes however it can be quicker to sketch out on paper what you are imagining it will look like. You can use thumbnail sketches to quickly detail the layout and visual hierarchy of your page.

Over the next few weeks we will add complexity to you designs
